# Arcana Oracle 🔮

Arcana Oracle est un site de tarot en ligne permettant de découvrir les messages des cartes du tarot.

Le site propose plusieurs tirages mystiques :

✨ Tarot du jour  
✨ Tirage passé – présent – futur  
✨ Tarot amour  
✨ Tarot travail  
✨ Tarot argent  
✨ Tarot oui / non  
✨ Tarot divinatoire  

Les cartes du tarot peuvent être tirées pour obtenir une guidance spirituelle et mieux comprendre les énergies du moment.

## Cartes du tarot

Le site contient les **78 cartes du tarot** avec leur signification :

- signification générale
- interprétation amour
- interprétation travail
- signification inversée

## Tirage

Les tirages utilisent un système de cartes aléatoires pour révéler les messages symboliques du tarot.
